Overview

For this task, I took the initiative to work on a Reinforcement Learning project to deepen my understanding of the subject, which I had studied in a university lecture.

In this project, we trained an agent to optimize the control of traffic signals at a junction. Instead of deploying a specific pre-defined algorithm, we trained the agent using a set of rewards. The algorithm used for this task was Deep SARSA.
